## Unravel
The goal of this project was to produce an "unraveling" effect where different shapes (i.e. circle, triangle, square) can be unraveled as if they were made of string.
The images are produced by drawing an outwardly growing spiral of a given shape and using perlin noise to vary the radius as the angle increases.
The various noise and size parameters were chosen so that the images produced are relatively similar and can be drawn and painted by hand to create the final artwork.
